# 设备连接向导

## 功能概述
提供更友好的设备连接向导，降低用户使用门槛。

## 用户场景
- 作为 **新用户**，我希望 **通过向导完成首次连接**，以便 **快速上手使用系统**

## 具体需求

### 需求 1：首次运行向导
- **描述**: 首次运行时自动弹出连接向导
- **优先级**: P0
- **向导步骤**:
  1. 欢迎页面 - 介绍系统功能
  2. 串口选择 - 列出可用串口
  3. 连接测试 - 测试串口通信
  4. 完成确认 - 显示连接成功/失败
- **预期结果**:
  - 首次启动时显示向导
  - 向导可跳过（保存设置后不再显示）

### 需求 2：高级连接设置
- **描述**: 在设置中提供高级串口参数配置
- **优先级**: P1
- **可配置项**:
  - 波特率（9600/19200/38400/115200）
  - 数据位（7/8）
  - 停止位（1/2）
  - 校验位（None/Odd/Even）
  - 流控制（RTS/CTS）
- **预期结果**:
  - 在设置中提供配置项
  - 提供常用预设（默认配置一键恢复）

### 需求 3：自动重连机制
- **描述**: 连接断开后自动尝试重连
- **优先级**: P1
- **功能要求**:
  - 断开后 3 秒自动重连
  - 最多重连 5 次
  - 重连失败后显示提示
  - 提供手动重连按钮
- **预期结果**:
  - 连接意外断开时自动尝试恢复
  - 重连状态有明确提示

## UI/UX 设计
- 向导使用 QWizard 实现
- 设置页面使用现有配置对话框
- 重连状态在状态栏显示

## 验收标准
- [ ] 首次运行向导正常引导用户
- [ ] 高级设置可以正确保存和加载
- [ ] 断开后自动重连功能正常
- [ ] 重连失败有明确提示

## 技术约束
- 向导状态保存到配置文件中
- 重连使用独立线程，不阻塞 UI

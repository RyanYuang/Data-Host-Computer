# 仪表盘增强功能

## 功能概述
为环境监测系统添加一个可视化仪表盘，提供更直观的数据展示和历史趋势分析。

## 用户场景
- 作为 **用户**，我希望 **在仪表盘上看到数据的可视化展示**，以便 **更直观地了解设备状态和环境变化**

## 具体需求

### 需求 1：实时仪表盘
- **描述**: 在主界面添加实时更新的仪表盘组件
- **优先级**: P0
- **UI 设计**:
  - 使用圆形仪表盘显示关键数据（温度、湿度）
  - 使用进度条显示设备状态
  - 使用指示灯显示连接状态
- **预期结果**: 
  - 温度以半圆仪表盘显示 0-50°C
  - 湿度以进度条显示 0-100%
  - CO 浓度以数值和颜色警示显示

### 需求 2：历史趋势图表
- **描述**: 添加数据历史趋势显示功能
- **优先级**: P1
- **功能要求**:
  - 记录过去 1 小时的数据点
  - 使用折线图显示温度/湿度变化趋势
  - 支持时间范围选择（1小时/6小时/24小时）
- **技术约束**: 使用 PyQt6 的 QCharts 或 matplotlib

### 需求 3：数据统计卡片
- **描述**: 在仪表盘区域显示关键统计数据
- **优先级**: P1
- **显示内容**:
  - 最高/最低温度
  - 平均湿度
  - 数据更新频率
  - 设备在线时长

## UI/UX 设计
- 仪表盘区域位于主界面顶部
- 历史图表在下方展开
- 统计卡片在图表旁边

## 验收标准
- [ ] 实时仪表盘正确显示当前数据
- [ ] 历史趋势图表能显示过去数据
- [ ] 统计卡片显示正确数据
- [ ] 界面布局合理美观

## 技术约束
- 使用 PyQt6 Charts 或 matplotlib
- 数据刷新率 ≤ 1秒
- 历史数据存储 ≤ 1000 个数据点

# 主题与个性化设置

## 功能概述
支持应用主题切换和个性化配置。

## 用户场景
- 作为 **用户**，我希望 **切换应用主题**，以便 **在不同环境下使用舒适的界面配色**
- 作为 **用户**，我希望 **自定义界面布局**，以便 **满足个人使用习惯**

## 具体需求

### 需求 1：深色/浅色主题
- **描述**: 支持深色和浅色主题切换
- **优先级**: P1
- **主题方案**:
  - **浅色主题**: 白色背景，蓝色强调色（当前默认）
  - **深色主题**: 深灰色背景，浅色文字
  - **跟随系统**: 跟随操作系统主题设置
- **预期结果**:
  - 在设置中可以切换主题
  - 切换后立即生效
  - 重启后保持上次选择的主题

### 需求 2：界面布局自定义
- **描述**: 允许用户调整组件位置和大小
- **优先级**: P2
- **功能要求**:
  - 可折叠的面板区域
  - 组件位置可拖拽调整
  - 保存布局到配置文件
- **预期结果**:
  - 用户可以收起/展开控制面板
  - 布局变化保存到配置文件

### 需求 3：语言设置
- **描述**: 支持中英文界面切换
- **优先级**: P2
- **支持语言**:
  - 简体中文（当前默认）
  - 英文
- **预期结果**:
  - 在设置中可切换语言
  - 切换后界面文字更新

## UI/UX 设计
- 设置页面增加"外观"标签页
- 提供主题预览缩略图
- 语言选择使用下拉框

## 验收标准
- [ ] 主题切换功能正常
- [ ] 重启后主题设置保持
- [ ] 界面布局可以自定义
- [ ] 语言切换生效

## 技术约束
- 主题样式使用 QSS（Qt Style Sheets）
- 布局配置保存到 JSON 文件
- 翻译使用字典映射
